# Review Understanding

Read the actual diff from this conversation. Present a scannable summary:

1. **What changed** — files and purpose of each change
2. **Key decisions** — choices that could have gone differently and why
3. **Implications** — new patterns, capabilities, or constraints going forward
4. **Watch-outs** — limitations, assumptions, things to revisit

Ask: **"Does this match your understanding, or is anything surprising?"** Address surprises.

Then ask 3-5 comprehension questions calibrated to complexity ("Why X not Y?", "What breaks if...?", "Where would you start to add Z?"). Review answers — confirm, clarify gaps, or explain without judgment.

Ask: **"Any questions?"** Answer directly.

End with a 2-3 sentence summary of where the project stands and the natural next step.
